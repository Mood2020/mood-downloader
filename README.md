# mood-downloader

دانلود ویدیو از یوتیوب با **GitHub Actions** و ذخیره داخل پوشه `videos/`.

## کاری که انجام می‌دهد
- از تب **Actions** لینک یوتیوب می‌گیری.
- با runnerهای خود گیت‌هاب ویدیو دانلود می‌شود.
- خروجی داخل `videos/` ذخیره می‌شود.
- بعد از دانلود، فایل‌ها ZIP می‌شوند.
- اگر فایل بزرگ باشد، ZIP به صورت پارت‌های `z01`, `z02`, ... و فایل نهایی `.zip` ساخته می‌شود.

## استفاده
1. پروژه را روی گیت‌هاب Push کن.
2. برو به تب **Actions**.
3. ورک‌فلو `Download YouTube to videos/` را انتخاب کن.
4. روی **Run workflow** بزن.
5. مقدارها را پر کن:
   - `youtube_url`: لینک ویدیو
   - `max_part_size_mb`: سایز هر پارت ZIP (مثلاً `1900`)
     - اگر `0` بدی، ZIP تکه‌تکه نمی‌شود.
6. بعد از اجرا، ورک‌فلو تغییرات را commit و push می‌کند.

## استخراج ZIP چندپارتی
وقتی `videos_bundle.z01`, `videos_bundle.z02`, ... و `videos_bundle.zip` را کنار هم داشته باشی:

- با 7-Zip / WinRAR روی `videos_bundle.zip` اکسترکت کن.
- برنامه به‌صورت خودکار پارت‌ها را یکی می‌کند.

## نکته
- این پروژه از `yt-dlp` استفاده می‌کند.
- دانلود محتوا باید مطابق قوانین یوتیوب و قوانین کپی‌رایت کشور شما باشد.
